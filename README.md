# To-Do-List_DOM
# Lista de Tarefas (To-Do List)

Aplicação web simples para gerenciamento de tarefas, desenvolvida com foco em manipulação do DOM utilizando JavaScript puro.

## Descrição

Este projeto consiste em uma lista de tarefas interativa que permite ao usuário adicionar, concluir, filtrar e remover tarefas. A aplicação também apresenta estatísticas em tempo real sobre o estado das tarefas.

## Funcionalidades

* Adição de novas tarefas
* Marcação de tarefas como concluídas
* Remoção de tarefas
* Contadores dinâmicos:

  * Total de tarefas
  * Tarefas concluídas
  * Tarefas pendentes
* Filtros de visualização:

  * Todas
  * Pendentes
  * Concluídas
* Alerta para grande volume de tarefas
* Limpeza de tarefas concluídas

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (manipulação do DOM)

## Objetivo

O objetivo deste projeto é consolidar conhecimentos fundamentais em desenvolvimento front-end, com ênfase em:

* Manipulação de elementos do DOM
* Tratamento de eventos
* Atualização dinâmica da interface
* Estruturação de código em aplicações web simples
